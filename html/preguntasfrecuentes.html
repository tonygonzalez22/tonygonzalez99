preguntas mas frecuentes

1. ¿Qué tipos de sistemas de seguridad ofrecen?

Ofrecemos una variedad de sistemas de seguridad, incluidos sistemas de cámaras de videovigilancia (CCTV), alarmas antirrobo, sistemas de control de acceso y domótica para el hogar inteligente.

2. ¿Cómo puedo instalar los sistemas de seguridad?

Nuestro equipo ofrece tanto opciones de instalación profesional como guías paso a paso para que puedas realizar la instalación por tu cuenta de manera sencilla.

3. ¿Qué sucede si mi sistema de seguridad detecta una intrusión?

Si tu sistema detecta una intrusión, recibirás una alerta instantánea en tu dispositivo móvil, y si el sistema está conectado a un servicio de monitoreo, las autoridades locales serán notificadas de inmediato.

4. ¿Puedo monitorear mi sistema de seguridad desde mi teléfono?

Sí, todos nuestros sistemas incluyen aplicaciones móviles que te permiten monitorear en tiempo real, recibir notificaciones y controlar el sistema desde cualquier lugar.

5. ¿Ofrecen servicio de soporte técnico?

Ofrecemos soporte técnico 24/7 para todos nuestros clientes, tanto por teléfono como por correo electrónico. También contamos con tutoriales y guías en línea.

6. ¿Qué garantía tienen los sistemas de seguridad?

Todos nuestros productos tienen una garantía de 2 años, que cubre tanto defectos de fabricación como problemas de funcionamiento.
